An enterprise platform for Real Estate Professionals.

Enabler:

  • Synchronize BLOBs
  • Mechanize backups
  • Deliver features continuously

Capabilities:

  • Blob → Blob
  • Blob → Folder
  • Blob → S3
  • Folder → Blob
  • Folder → Folder
  • Folder → S3
  • S3 → Blob
  • S3 → Folder
  • S3 → S3

Use Case:

Being able to easily push JavaScript/CSS/Images to Blob Storage instead of having to redeploy an Azure package.

Use Case:

Being able to hotfix a production web application by turning off a feature via JavaScript or CSS instead of having to redeploy.

Use Case:

Being able to commit to master and have your build server run the tests... have those tests pass and then push features into production automatically.

Use Case:

Being able to automate backups (pushing Azure packages or DACPACs to Blob Storage).
